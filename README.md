# Фото как кино

Сайт-визитка фотографа Ксении Аржанниковой.

Проект создается как легкий, быстрый и аккуратный сайт без CMS и лишней сложности. Контент будет храниться прямо в коде, изображения — в `public`.

## Стек

- Next.js
- TypeScript
- CSS Modules
- `next/image`
- `next/font`
- React Hook Form
- Zod

## Запуск

```bash
pnpm install
pnpm dev
```

Сайт откроется на `http://localhost:3000`.

## Проверка

```bash
pnpm lint
pnpm build
```

## Структура

Код пишем просто и читаемо. Архитектура — FSD:

- `app` — страницы и layout;
- `widgets` — крупные секции страницы;
- `features` — действия пользователя;
- `entities` — данные предметной области;
- `shared` — общие настройки, утилиты и UI.

## Лицензия
