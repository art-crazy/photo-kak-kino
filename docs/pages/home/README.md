# Главная страница

`final.html` — исходный HTML-макет главной страницы.

Использовать как визуальный и смысловой источник при переносе в Next.js:

- верхнее меню: «Портфолио», «Условия съемки», «Контакты»;
- изображения брать из `public/site-photos`;
- код реализации писать на FSD, просто и читаемо;
- макет не усложнять без явной необходимости.
